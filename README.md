# 📊 Projeto de Análise de Dados

## 📌 Visão Geral
Este projeto tem como objetivo realizar um processo completo de análise de dados, desde a importação e limpeza dos dados até a criação de dashboards e relatórios visuais. O projeto utiliza Python para análise exploratória (EDA), SQL para consultas em banco de dados e Power BI para visualização dos insights.

---

## 📂 Dataset
O conjunto de dados utilizado contém informações relacionadas ao comportamento de clientes, compras, avaliações de produtos e métodos de envio.

### Principais colunas:
- Customer ID
- Gender
- Age
- Purchase Amount
- Item Purchased
- Review Rating
- Shipping Type
- Subscription Status
- Discount Applied

## 🔎 Etapas do Projeto

### 1. Importação dos Dados
Os dados foram carregados utilizando Python e armazenados para análise.

### 2. Limpeza dos Dados
Foram realizados:
- Tratamento de valores nulos
- Remoção de duplicatas
- Padronização de colunas
- Conversão de tipos de dados

### 3. Análise Exploratória (EDA)
Durante a análise exploratória foram identificados:
- Produtos mais vendidos
- Média de avaliações
- Perfil dos clientes
- Impacto dos descontos
- Comparação entre tipos de envio

### 4. Consultas SQL
Consultas realizadas em PostgreSQL/MySQL/SQL Server para extrair insights como:
- Receita total por gênero
- Top produtos por avaliação
- Clientes com maior gasto
- Comparação entre assinantes e não assinantes
- Análise de descontos e receita

### 5. Dashboard no Power BI
Foi desenvolvido um dashboard interativo contendo:
- KPIs principais
- Gráficos de vendas
- Distribuição de clientes
- Receita por categoria
- Comparações de desempenho

### 6. Relatório e Apresentação
Os resultados foram documentados em:
- Relatório analítico
- Apresentação em PPT criada no Gamma

---

## 📊 Principais Resultados
- Identificação dos produtos com melhor avaliação
- Descoberta dos grupos de clientes com maior receita
- Comparação entre usuários assinantes e não assinantes
- Impacto do frete expresso no valor médio das compras
- Insights para estratégias de marketing e fidelização

## ▶️ Como Executar o Projeto

### 1. Clone o repositório
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

### 2. Instale as dependências
```bash
pip install -r requirements.txt
```

### 3. Execute o notebook
Abra o Jupyter Notebook:
```bash
jupyter notebook
```

### 4. Banco de Dados
Configure o PostgreSQL/MySQL/SQL Server e execute os scripts SQL disponíveis na pasta do projeto.

### 5. Dashboard
Abra o arquivo `.pbix` no Power BI Desktop.
